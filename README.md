В это же время я начал проходить хендбук «Базовые алгоритмы на С++» в Яндекс LMS. Позже, уже на втором курсе университета, произошло знакомство со структурами и алгоритмами обработки данных. Рассмотрел основные алгоритмы поиска и сортировки. работу со списками разных типов.
